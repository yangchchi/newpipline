## 3. API
### 3.1 POST /api/v1/projects
- 请求：
```json
{
  "input_mode": "script",
  "content": "# Market\nContent here.",
  "template_id": "tech_blue",
  "display_mode": "landscape",
  "voice_id": "voice_male_01",
  "speed_rate": 1.0
}
```
- 响应：
```json
{
  "project_id": "proj_123",
  "status": "pending"
}
```
- 错误码：
    - 40001: 输入内容少于10字符
    - 40002: 内容包含敏感词
    - 50001: 内部服务错误

### 3.2 GET /api/v1/projects/:id
- 请求：路径参数 `id`
- 响应：
```json
{
  "project_id": "proj_123",
  "status": "success",
  "timeline_data": {...},
  "audio_url": "https://cdn.example.com/audio.mp3",
  "failed_reason": null
}
```
- 错误码：
    - 40401: 项目不存在

### 3.3 POST /api/v1/projects/:id/exports
- 请求：
```json
{
  "resolution": "1080p"
}
```
- 响应：
```json
{
  "task_id": "task_456",
  "status": "queued",
  "estimated_finish_at": "2024-05-27T10:05:00Z"
}
```
- 错误码：
    - 40301: 免费用户导出次数超限
    - 40401: 项目不存在
    - 50001: 内部服务错误

## 4. 核心流程
### 4.1 创建项目流程
```
1. 接收请求，解析参数
2. 校验内容长度 >= 10 字符
3. 调用敏感词服务校验内容
4. 根据 input_mode 解析结构化内容：
   - 脚本模式：按 H1 分页，H2 作为节
   - 大纲模式：按缩进层级解析
5. 创建 Project 记录，status=pending
6. 异步触发生成任务：
   go func() {
     a. 更新 status=processing
     b. 调用 TTS 服务生成音频 -> audio_url
     c. 计算时间轴 timeline_data
     d. 更新 status=success, 保存 audio_url, timeline_data
     e. 如果失败：更新 status=failed, failed_reason
   }()
7. 返回 project_id
```

### 4.2 导出视频流程
```
1. 接收请求，解析 resolution
2. 查询 UserQuota 表：
   if plan == "free" && export_count >= 5:
     返回错误 40301
3. 创建 ExportTask 记录，status=queued
4. 异步触发渲染任务：
   go func() {
     a. 更新 status=processing
     b. 调用视频渲染服务
     c. 生成 MP4 文件，上传到 CDN
     d. 更新 status=success, output_url
     e. 更新 UserQuota.export_count += 1
     f. 发送系统通知
     g. 如果失败：更新 status=failed
   }()
5. 返回 task_id
```

## 5. 异常处理
- 错误规则：
    - 所有业务错误返回标准错误响应：`{"code": "40001", "message": "..."}`
    - 使用 `context.Context` 传递请求上下文
    - 禁止使用 `panic()` 处理业务错误
    - 数据库操作使用 GORM 错误处理
    - 外部服务调用需设置超时和重试

```json
  "speed_rate": 1.0
```

- 输出：`{"code": "40002", "message": "内容包含违规词汇，请修改"}`

- 输入：`{"resolution": "1080p"}`
- 输出：`{"code": "40301", "message": "本月免费导出次数已用尽，请升级套餐或下月再试"}`
