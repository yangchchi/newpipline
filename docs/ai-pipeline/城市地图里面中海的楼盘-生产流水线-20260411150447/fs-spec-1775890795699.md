# FS 正文（Markdown）

# FS

## 1. 目标
解决投研人员需在多系统间切换以定位和查询中海楼盘信息，导致分析效率低下的问题。成功标准为：上线后1个月内，投研团队使用地图进行楼盘定位和分析的日均时长提升30%；单个楼盘的基础信息查询与定位平均耗时从5分钟降低至30秒以内。

## 2. 角色与场景
- 用户：投研分析师、投研总监
- 使用场景：区域市场扫描、特定项目查询、竞品对比准备

## 3. 功能
### 功能点1：图层开关控制
- 输入：用户点击“中海楼盘”图层开关按钮
- 规则：
    - IF 用户点击开关且当前状态为“关闭” THEN 发送请求获取当前地图视野内的楼盘数据，开关状态变为“开启”
    - IF 用户点击开关且当前状态为“开启” THEN 清除地图上所有中海楼盘图标，开关状态变为“关闭”
    - IF 数据请求失败 THEN 开关置灰不可用，并显示Toast提示：“数据加载失败，请稍后重试”
- 输出：
    - 开关状态变化（开启/关闭）
    - 地图上对应图层的显示/隐藏
- 验收标准：
    - 点击开关，地图正确显示或隐藏所有中海楼盘图标。
    - 数据接口异常时，开关置灰并给出明确错误提示。

### 功能点2：楼盘图标渲染与悬停
- 输入：
    1. 图层开启后，系统获取的楼盘数据列表（包含经纬度坐标、状态、案名）
    2. 用户鼠标悬停在楼盘图标上
- 规则：
    - IF 获取到楼盘数据列表 THEN 根据每个楼盘的`状态`字段，在地图对应坐标渲染预设图标（如：绿色-在售，灰色-售罄，黄色-在建）
    - IF 当前地图缩放级别小于预设阈值（如：小于10级）且多个楼盘坐标距离过近 THEN 使用聚合点图标代替
    - IF 用户鼠标悬停在单个楼盘图标上 THEN 该图标视觉上放大或颜色加深，并显示包含楼盘`案名`的Tooltip
    - IF 用户鼠标离开图标 THEN Tooltip消失，图标恢复原状
- 输出：
    - 地图上渲染出带状态的楼盘图标或聚合点
    - 鼠标悬停时显示对应楼盘的案名Tooltip
- 验收标准：
    - 不同状态的楼盘图标样式正确区分。
    - 鼠标悬停时，Tooltip正确显示楼盘案名。

### 功能点3：楼盘搜索
- 输入：用户在搜索框输入楼盘案名的关键字
- 规则：
    - IF 用户输入字符 THEN 实时调用接口，对`案名`字段进行模糊匹配
    - IF 接口返回匹配结果 THEN 在下拉列表中显示最多10条结果（格式：案名 - 区域）
    - IF 用户点击下拉列表中的一条结果 THEN 地图平滑移动并缩放到该楼盘坐标，高亮该楼盘图标，并自动弹出其详情弹窗（触发功能点5）
    - IF 搜索框为空或用户清空输入 THEN 隐藏下拉列表
- 输出：
    - 实时显示匹配的搜索建议下拉列表
    - 地图定位、高亮并展示目标楼盘详情
- 验收标准：
    - 输入“寰宇天下”，下拉列表出现“中海·寰宇天下 - A市高新区”。
    - 点击该结果，地图定位到该项目并弹出详情。

### 功能点4：楼盘状态筛选
- 输入：用户在筛选面板中选择一个或多个“楼盘状态”（多选）
- 规则：
    - IF 用户点击“应用筛选”按钮 THEN 将所选状态列表（如`[“在售”, “待售”]`）作为参数请求楼盘数据
    - IF 接口返回数据 THEN 地图上仅渲染符合筛选状态的楼盘图标
    - IF 用户点击“重置筛选”按钮 THEN 清空所有已选状态，地图渲染所有楼盘图标
- 输出：
    - 地图上仅显示符合筛选状态的楼盘图标
- 验收标准：
    - 筛选“在售”状态，地图上只显示在售楼盘图标。
    - 点击“重置”，所有楼盘图标恢复显示。

### 功能点5：楼盘详情查看
- 输入：用户点击地图上的楼盘图标
- 规则：
    - IF 用户点击楼盘图标 THEN 在该图标旁弹出固定大小的模态弹窗
    - IF 弹窗弹出 THEN 弹窗内结构化展示该楼盘的以下字段：案名、所属城市/区域、楼盘状态、主力户型、最新均价、项目地址、数据更新时间
    - IF 用户点击弹窗右上角的“X”按钮或点击地图空白区域 THEN 关闭弹窗
    - IF 用户拖动或缩放地图 THEN 自动关闭已打开的弹窗
- 输出：
    - 显示包含楼盘详细信息的弹窗
- 验收标准：
    - 点击任意楼盘图标，弹出其详情弹窗，信息完整正确。
    - 点击弹窗外部或地图操作时，弹窗正确关闭。

## 4. 规则补充
- 全局业务规则（if-then）
    - IF 当前地图视野内无符合条件（如无数据、筛选后无结果）的楼盘 THEN 不显示任何图标，并在图层开关旁显示文字提示：“当前区域无数据”
    - IF 用户操作时（如点击、搜索）网络断开 THEN 显示Toast提示：“网络连接已断开”
    - IF 用户未登录或无“投研地图”访问权限 THEN 禁止访问地图页面，跳转至登录页或权限提示页

## 5. 示例（必须）
### 正常：
Input: 用户开启“中海楼盘”图层，地图视野位于A市高新区。
Output: 地图上渲染出3个图标：2个绿色（在售）图标，1个灰色（售罄）图标。

Input: 用户在搜索框输入“寰宇”。
Output: 下拉列表显示“中海·寰宇天下 - A市高新区”。用户点击后，地图定位并高亮该项目，弹出详情弹窗显示其案名、状态、户型等信息。

### 异常：
Input: “中海楼盘”图层开启时，数据接口请求超时。
Output: 图层开关置灰，页面显示Toast提示：“数据加载失败，请稍后重试”。

Input: 用户筛选“在售”状态，但当前地图视野内无在售楼盘。
Output: 地图上无任何楼盘图标显示，图层开关旁显示文字提示：“当前区域无数据”。
