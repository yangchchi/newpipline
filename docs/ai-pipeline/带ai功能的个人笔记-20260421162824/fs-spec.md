# FS 正文（Markdown）

# FS

## 1. 目标
开发一款集成AI能力的个人笔记应用，通过智能整理、内容辅助和智能检索功能，解决用户“记了难找、记了难用”的核心问题，提升从记录到应用的全流程效率。
**成功标准**：
1.  AI功能平均响应时间<2秒。
2.  核心操作路径（创建、编辑、搜索）用户任务完成率>95%。
3.  上线12个月内，实现累计注册用户100万，MAU 30万。

## 2. 角色与场景
- **用户**：学生、职场人士、知识工作者（作家、研究员等）。
- **使用场景**：课堂/会议笔记整理、报告/论文撰写、主题研究与知识管理。

## 3. 用户故事
### 用户故事1：作为用户，我希望在保存笔记时，能自动获得内容标签、摘要和分类建议，以便快速整理归档。
#### 验收标准：
##### （1）用户在编辑器页点击“保存”按钮。
##### （2）如果笔记内容字符数>10，则调用【智能整理】服务。
##### （3）服务返回：3-5个标签建议、一段≤150字的摘要、1-3个分类文件夹推荐。
##### （4）前端以非模态面板（从底部弹出）展示所有建议，用户可进行“一键采纳”、“选择性采纳”或“忽略”操作。
##### （5）如果笔记内容字符数≤10，则不调用【智能整理】服务，仅保存笔记。
##### （6）如果调用【智能整理】服务超时（>3秒），则保存笔记，并显示Toast提示：“AI服务暂不可用，笔记已保存”。

### 用户故事2：作为用户，我在编辑笔记时，希望获得选中文本的写作优化建议或语法修正，以提升内容质量。
#### 验收标准：
##### （1）用户在编辑器页选中一段文本，点击工具栏的“AI助手”图标。
##### （2）前端弹出菜单，显示“改进写作”和“检查语法”选项。
##### （3）如果用户点击“改进写作”，则调用【内容辅助-写作建议】服务。
##### （4）服务返回1-3条具体的修改建议文本。
##### （5）前端以列表形式展示建议，用户点击某条建议可预览替换效果，并选择“替换”或“取消”。
##### （6）如果用户点击“检查语法”，则调用【内容辅助-语法修正】服务（优先使用本地轻量模型）。
##### （7）服务返回文本中的错误位置及修正建议。
##### （8）前端以红色波浪下划线标出错误，用户点击下划线可查看建议并选择“修正”或“忽略”。

### 用户故事3：作为用户，我希望使用自然语言搜索我的所有笔记，而不仅是关键词匹配，以便快速找到所需内容。
#### 验收标准：
##### （1）用户在首页顶部搜索框输入查询语句（如：“上周开会关于产品定价的结论”）。
##### （2）前端实时（或点击搜索按钮后）调用【智能检索-自然语言搜索】服务。
##### （3）服务基于语义理解，返回按相关性排序的笔记列表。
##### （4）前端展示搜索结果列表，每条结果包含：笔记标题、匹配内容片段、所属笔记本、最后修改日期。
##### （5）如果用户未输入任何字符，则显示“最近搜索记录”或“热门标签”。

### 用户故事4：作为用户，我希望在离线状态下也能创建、编辑和查看笔记，所有更改能在网络恢复后自动同步。
#### 验收标准：
##### （1）用户在网络断开时，打开应用。
##### （2）所有笔记的本地操作（增、删、改、查）功能均正常可用。
##### （3）如果用户尝试使用依赖网络的AI功能（如：智能整理、灵感扩展），则显示提示：“该功能需要网络连接”。
##### （4）当网络恢复时，应用自动将本地待同步数据加入队列，并在30秒内完成增量同步。
##### （5）如果同步时检测到多设备修改同一笔记的冲突，则提示用户，并提供选项：“查看差异并手动合并”或“以最新版本为准”。

## 4. 规则补充
### 全局业务规则（if-then）
1.  **AI内容标识规则**：
    -   IF 笔记中的内容段落由AI生成（如：采纳的写作建议、插入的灵感扩展）或AI显著修改。
    -   THEN 该段落需添加视觉标识（如：浅灰色背景、或段落末尾显示“AI生成”图标）。
2.  **数据加密规则**：
    -   IF 笔记数据（标题、正文、标签）需要存储到本地或向服务器同步。
    -   THEN 必须使用用户设备生成并管理的密钥进行端到端加密，服务器仅存储加密后的密文。
3.  **隐私保护规则**：
    -   IF 需要收集使用数据以改进产品（如：功能点击率）。
    -   THEN 必须事先获得用户明确同意，且数据必须匿名化，严禁将用户笔记原始内容用于任何模型训练。
4.  **笔记删除规则**：
    -   IF 用户删除一条笔记。
    -   THEN 笔记被移至“废纸篓”。
    -   IF 笔记在“废纸篓”中留存时间超过30天。
    -   THEN 系统自动永久删除该笔记。
5.  **AI服务降级规则**：
    -   IF 调用任一AI服务（智能整理、内容辅助、智能检索）的响应时间超过5秒。
    -   THEN 前端终止等待，显示Toast提示：“请求超时，请稍后重试”，并保证基础笔记操作不受影响。
6.  **输入有效性规则**：
    -   IF 用户触发【智能整理】或【内容辅助】功能。
    -   AND 输入文本（笔记内容或选中文本）字符数≤10。
    -   THEN 不调用AI服务，并提示：“内容过短，无法提供AI建议”。
